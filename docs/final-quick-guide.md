# 决赛评分现场一分钟操作卡

更新日期：2026-07-14

## 管理员赛前

1. 登录后看页面顶部“当前阶段”，先处理红色阻塞项。
2. 在“队伍管理”核对报名编号、队伍名称、项目名称和抽签顺序。
3. 在“评委管理”确认账号状态；赛中增加评委默认只从下一支队伍生效。
4. 回到“开赛配置”，确认管理员密码、队伍资料、本场队伍、评委账号和本场评委。
5. 保存配置，再开启当天比赛组别。

## 每支队伍

1. 点击顶部建议动作，核对推荐队伍与现场路演队伍一致后派发。
2. 等待当前队评分快照内全部评委提交。
3. 第 1、2 支队伍完成后暂不发布，继续按抽签顺序评分；第 3 支完成后进入“前三队集中公布”，按第 1、2、3 支顺序展示成绩。
4. 展示到第 3 支后再派发第 4 支；从第 4 支起，每支队伍完成后核对综合分并立即展示。
5. 展示时间最终由管理员决定，系统只提供流程建议，不自动发布成绩。
6. 最后一队形成最终成绩后即可结束赛次；是否展示该队成绩由管理员决定，不构成关赛门禁。

## 评委

1. 登录后等待管理员派发，不选择组别或队伍。
2. 点击每项右侧大分数区，使用数字键盘录入两位小数。
3. 全部录完后提交；缺项时系统会定位第一项缺分。
4. 显示“服务器已确认”才算提交完成。
5. 若显示“当前页面暂存”，不要刷新、退出或重录，保持页面等待网络恢复。

## 应急入口

管理员统一进入“应急处置”：

| 现场问题 | 选择 | 影响范围 |
| --- | --- | --- |
| 误提交、单人重评、当前评委离场 | 当前队评委异常 | 一个评委在当前队的席位 |
| 当前队无法继续、必须切换队伍 | 强制切队 | 当前派发；必须填写原因 |
| 整组队伍数或评委数错误 | 重新配置当前组 | 清除本组评分和快照，保留资料、顺序、账号 |
| 下一队开始增加评委 | 评委管理 → 新增评委 | 仅后续首次派发队伍 |
| 停用或归档后续评委 | 评委管理 → 选择账号 → 停用/归档 | 同次保存移出后续计划名册；当前队成员须先替换 |

任何应急操作后，都要核对顶部当前阶段、当前队、评委提交矩阵和投屏队伍。

重新启用评委账号不会自动恢复参评资格；需要管理员明确将其加入后续计划评分名册。
